Para compilar y ejecutar el programa realice los siguientes pasos:

1. Abrir la carpeta del proyecto en el cmd (Símbolo del sistema).
2. Escriba "cd src" para cambiar de carpeta y presione "Enter".
3. Para compilar escriba en la línea de comando "javac test/DELinkListTest.java" y presione "Enter".
5. Despúes, para ejecutar escriba el comando "java test/DELinkListTest" y presionar "Enter".
